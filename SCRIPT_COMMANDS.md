# 自动化脚本命令参考

## 使用方法

### 方式1: 从文件执行
```python
from script_runner import run_file

run_file("example_script.txt")
```

### 方式2: 直接运行
```bash
python script_runner.py example_script.txt
```

### 方式3: 交互模式
```bash
python script_runner.py
```

---

## 支持的命令

### 📋 窗口操作

| 命令 | 示例 | 说明 |
|------|------|------|
| `找窗口:标题` | `找窗口:WorkBuddy` | 切换到指定窗口 |
| `切换窗口:标题` | `切换窗口:Notepad` | 同上 |
| `窗口:标题` | `窗口:Chrome` | 同上 |

### 🖱️ 图片点击

| 命令 | 示例 | 说明 |
|------|------|------|
| `点击图片:名称` | `点击图片:自动化` | 在窗口中查找并点击图片 |
| `点击:名称` | `点击:登录` | 同上 |
| `右键点击图片:名称` | `右键点击图片:菜单` | 右键点击图片 |
| `双击图片:名称` | `双击图片:文件夹` | 双击图片 |

> 注意：图片名称会自动添加 `.png` 后缀
>
> **图片搜索路径**（按顺序查找）：
> 1. 当前工作目录（`./自动化.png`）
> 2. `images/` 子目录（`./images/自动化.png`）
> 3. 脚本所在目录
> 4. 脚本所在目录的 `images/` 子目录

**推荐做法**：把图片模板都放在 `images/` 文件夹，保持项目整洁

### 🖱️ 鼠标操作

| 命令 | 示例 | 说明 |
|------|------|------|
| `鼠标移动到:x,y` | `鼠标移动到:100,200` | 移动到绝对坐标 |
| `移动到:x,y` | `移动到:0,0` | 同上 |
| `鼠标移动:x,y` | `鼠标移动:+50,-30` | 相对当前位置移动 |
| `相对移动:x,y` | `相对移动:-10,+20` | 同上 |
| `点击` | `点击` | 点击当前位置 |
| `右键` | `右键` | 右键点击当前位置 |
| `双击` | `双击` | 双击当前位置 |
| `滚轮:数量` | `滚轮:3` | 滚动（正=上，负=下） |

### ⌨️ 键盘操作

| 命令 | 示例 | 说明 |
|------|------|------|
| `输入:文本` | `输入:Hello` | 输入文本 |
| `打字:文本` | `打字:密码123` | 同上 |
| `按键:键名` | `按键:enter` | 按一次键 |
| `按住:键名` | `按住:ctrl` | 按住按键 |
| `释放:键名` | `释放:ctrl` | 释放按键 |

**常用按键名**：
- `enter`, `tab`, `esc`, `space`
- `backspace`, `delete`
- `ctrl`, `alt`, `shift`, `win`
- `up`, `down`, `left`, `right`
- `home`, `end`, `pageup`, `pagedown`
- 组合键：`ctrl+c`, `alt+tab`, `ctrl+v`

### ⏳ 等待与控制

| 命令 | 示例 | 说明 |
|------|------|------|
| `等待:秒` | `等待:2` | 等待指定秒数 |
| `等待:1.5` | `等待:1.5` | 支持小数 |
| `截图:文件名` | `截图:shot.png` | 保存截图 |

### ⚙️ 设置

| 命令 | 示例 | 说明 |
|------|------|------|
| `置信度:0.9` | `置信度:0.85` | 设置图片匹配精度 |
| `调试:开` | `调试:on` | 开启调试输出 |
| `调试:关` | `调试:off` | 关闭调试输出 |

### 📝 注释

| 命令 | 示例 | 说明 |
|------|------|------|
| `# 注释` | `# 这是注释` | 注释（被忽略） |
| `// 注释` | `// TODO` | 同上 |

## 示例脚本

```text
# 打开 WorkBuddy 并点击图片

找窗口:WorkBuddy
点击图片:自动化
等待:0.5
点击图片:确定
等待:1
输入:Hello World
按键:enter
```

## 错误处理

- 命令执行失败会显示 `❌` 并继续执行下一行
- 图片找不到会打印警告但不会中断脚本
- 使用 `run()` 或 `run_file()` 返回结果统计

```python
result = run_file("script.txt")
print(f"成功: {result['success_count']}, 失败: {result['fail_count']}")
```
